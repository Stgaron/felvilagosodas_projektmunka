# A munka menete
Alakítsatok csapatokat (2-3 fő) Egyvalaki forkolja le ez a repot, és hívja meg a csapattagokat. Mindenki clone-ozza le a közös repót. (csillagozd meg!) Mostantól ebben dolgoztok!

A feladat megoldása 5 sprintben történik.

### 1. sprint - Tervezés

- Feladatok megbeszélése és rögzítése a Plannerben
- Anyaggyűjtés
- Vázlat készítése
- A szövegeket txt állományba helyezzétek el a repóban, a képeket külön mappába gyűjtsétek. Kategorizáljátok, a mappákban is hozhattok létre újabb mappákat a rendszerezéshez.

###  2. sprint - Tervezés
Szövegek megírása
Képek keresése / készítése

**Tartalomfelelős** – történelmi szövegek **ellenőrzése** (tartalmi mennyiségi és minőségi, helyesírás)
**Kvíz-felelős** – fogalmak összegyűjtése
**Vizualizáció-felelős** – képek, ábrák ellenőrzése, rendeszerezése, minden témához van-e kép? Források pontosítása, feljegyzése
